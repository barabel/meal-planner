# InputWrapper

Обёртка для полей ввода с поддержкой label и отображения ошибки.

## Props

| Prop        | Тип       | Описание                        |
|-------------|-----------|-------------------------------|
| `label`     | `string?` | Подпись над полем               |
| `error`     | `string?` | Текст ошибки (показывает иконку с тултипом) |
| `className` | `string?` | Дополнительные CSS-классы       |
| `children`  | `ReactNode` | Поле ввода или другой контент |

## Поведение

- При наличии `error` добавляется класс `error-input` на корневой элемент и появляется иконка-индикатор
- Тултип с текстом ошибки показывается при hover на иконку

## Использование

```tsx
<InputWrapper label="Название" error="Обязательное поле">
  <Input />
</InputWrapper>
```
